## Chapter 2 App.java , Calculator.java
- 콘솔 메뉴를 통해 원하는 동작을 입력한다 (1.계산하기 2.계산내역보기 3.마지막계산기록삭제)
- 기존 App.java에서 계산 -> Calculator.java 클래스에서 계산하도록 수정

### 1.계산하기
- 첫번째 양의 정수를 입력한다.
- 양식에 맞는 연산자 기호를 입력한다 ( + , - , * , /)
- 두번째 양의 정수를 입력한다.
- 계산결과를 확인한다
- Scanner에 "exit" 문자열을 입력받을시 콘솔메뉴로 빠져나간다

### 2.계산결과확인
- 계산했던 결과들을 출력해준다 
ex) "x + y = result"
- 저장된 계산기록 데이터가없을경우 메세지출력
- Scanner에 값을 입력받을시 콘솔메뉴로 빠져나온다.

### 3.마지막계산기록삭제
- 계산기록들중 index 0번 기록을 삭제한다
- 저장된 계산기록 데이터가없을경우 메세지출력
-  Scanner에 값을 입력받을시 콘솔메뉴로 빠져나온다.

## Chater 3 App.java , ArithemticCalculator.java , Operation.java

- 콘솔 메뉴를 통해 원하는 동작을 입력한다 (1.계산하기 2.계산내역보기 3.선택계산기록삭제)
- App.java = 입력,출력,유효성검사
- ArithemticCalculator = 연산기록저장, 연산기록삭제, 저장된연산결과중 결과값보다 더 높은값출력
- Operation = 입력된 연산자 확인후 그에 맞는 연산메소드 제공

### 1.계산하기
- 첫번째 숫자를 입력한다.(double)
- 양식에 맞는 연산자 기호를 입력한다 ( + , - , * , /)
- 두번째 숫자를 입력한다.(double)
- 계산결과를 확인한다
- 입력된 결과값보다 더 높은수를 가진 결과값을 저장된 결과데이터에서 출력시킨다
- Scanner에 "exit" 문자열을 입력받을시 콘솔메뉴로 빠져나간다

### 2.계산결과확인
- 계산했던 결과들을 출력해준다
  ex) "x + y = result"
  저장된 계산기록 데이터가없을경우 메세지출력
- Scanner에 값을 입력받을시 콘솔메뉴로 빠져나온다.

### 3.마지막계산기록삭제
- 계산기록들중 삭제하고싶은 index를 선택해 삭제한다
- 저장된 계산기록 데이터가없을경우 메세지출력
- Scanner에 0을 입력받을시 콘솔메뉴로 빠져나온다.
