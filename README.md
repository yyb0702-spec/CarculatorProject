# 🧮 계산기 프로젝트

## 📌 소개

Java를 사용하여 만든 계산기입니다.
사칙연산을 기반한 계산,
계산 기록 저장,
계산 기록 삭제,
계산결과 보다 더 높은수의 값 출력, 기능을 포함하고 있습니다.

## ⚙️ 기능
- 사칙연산 (+, -, *, /) 을 이용한 계산
- 계산 결과 저장
- 계산 기록 조회
- 계산 값보다 큰 결과 출력 (Stream 사용)
- 계산 기록 삭제

## 🛠️ 기술 스택

- Java
- Stream
- Enum
- List (ArrayList)

## 📂 구조
### Chapter 1 App.java
- 첫번째 양의 정수를 입력한다.
- 양식에 맞는 연산자 기호를 입력한다 ( + , - , * , /)
- 두번째 양의 정수를 입력한다.
- 계산결과를 확인한다
- Scanner에 "exit" 문자열을 입력받을시 종료한다

### Chapter 2 App.java , Calculator.java
- 콘솔 메뉴를 통해 원하는 동작을 입력한다 (1.계산하기 2.계산내역보기 3.마지막계산기록삭제)
- 기존 App.java에서 계산 -> Calculator.java 클래스에서 계산하도록 수정

#### 1.계산하기
- 첫번째 양의 정수를 입력한다.
- 양식에 맞는 연산자 기호를 입력한다 ( + , - , * , /)
- 두번째 양의 정수를 입력한다.
- 계산결과를 확인한다
- Scanner에 "exit" 문자열을 입력받을시 콘솔메뉴로 빠져나간다

#### 2.계산결과확인
- 계산했던 결과들을 출력해준다 
ex) "x + y = result"
- 저장된 계산기록 데이터가없을경우 메세지출력
- Scanner에 값을 입력받을시 콘솔메뉴로 빠져나온다.

#### 3.마지막계산기록삭제
- 계산기록들중 index 0번 기록을 삭제한다
- 저장된 계산기록 데이터가없을경우 메세지출력
-  Scanner에 값을 입력받을시 콘솔메뉴로 빠져나온다.

### Chater 3 App.java , ArithemticCalculator.java , Operation.java

- 콘솔 메뉴를 통해 원하는 동작을 입력한다 (1.계산하기 2.계산내역보기 3.선택계산기록삭제)
- App.java = 입력,출력,유효성검사
- ArithemticCalculator = 연산기록저장, 연산기록삭제, 저장된연산결과중 결과값보다 더 높은값출력
- Operation = 입력된 연산자 확인후 그에 맞는 연산메소드 제공

#### 1.계산하기
- 첫번째 숫자를 입력한다.(double)
- 양식에 맞는 연산자 기호를 입력한다 ( + , - , * , /)
- 두번째 숫자를 입력한다.(double)
- 계산결과를 확인한다
- 입력된 결과값보다 더 높은수를 가진 결과값을 저장된 결과데이터에서 출력시킨다
- Scanner에 "exit" 문자열을 입력받을시 콘솔메뉴로 빠져나간다

#### 2.계산결과확인
- 계산했던 결과들을 출력해준다
  ex) "x + y = result"
  저장된 계산기록 데이터가없을경우 메세지출력
- Scanner에 값을 입력받을시 콘솔메뉴로 빠져나온다.

#### 3.마지막계산기록삭제
- 계산기록들중 삭제하고싶은 index를 선택해 삭제한다
- 저장된 계산기록 데이터가없을경우 메세지출력
- Scanner에 0을 입력받을시 콘솔메뉴로 빠져나온다.

- enum 객체를 생성시켜 Switch문을 대신해 더 간결하게 사용할수있는걸 배웠습니다
- Stream을 활용해 조건 필터링을 구현하면서 람다식에 대해 공부하면서 여러 명령어들을 알게되었습니다.
- 예외처리를 통해 프로그램이 어떤 예외상황일때 멈추는지 알게되고, 클라이언트 입력 안정성을 높이는 방법을 경험했습니다.
